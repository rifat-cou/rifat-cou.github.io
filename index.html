Line Follower Robot — Autonomous Embedded System
Microprocessor & Microcontroller Course · 2023–2024
Autonomous robot following a black-line path using 5-array IR sensor feedback and Arduino Nano as central control unit. Evaluated ON/OFF vs. PID control — selected PID for smoother navigation with reduced oscillation at T-junctions. 3D-printed chassis with N20 DC gear motors (600+ RPM) and L298N motor driver, all motion logic in C++. Validated across 5+ real-world scenarios.
